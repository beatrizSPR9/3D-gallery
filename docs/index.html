Features

  • Navigable 3D gallery with interactive camera
  • Video playback on a 3D wall
  • Image display in 3D space
  • Media information panel
  • Face detection with OpenCV
  • XML-based configuration
  • Keyboard and mouse interaction

Technologies

C++ OpenFrameworks 0.12 OpenGL OpenCV (ofxCv) FMOD (audio) FreeImage Visual Studio 2022 XML (ofxXmlSettings)
